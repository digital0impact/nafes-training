# البيئة التقنية المستخدمة في التطبيق

## نظرة عامة

هذا التطبيق مبني باستخدام أحدث التقنيات الحديثة لتطوير تطبيقات الويب.

## إطار العمل الرئيسي

### Next.js
- **الإصدار:** 14.2.33
- **الوصف:** إطار عمل React للواجهات الأمامية والخلفية
- **الاستخدام:** بناء صفحات التطبيق، API Routes، Server Components

### React
- **الإصدار:** 18.3.1
- **الوصف:** مكتبة JavaScript لبناء واجهات المستخدم التفاعلية
- **الاستخدام:** مكونات الواجهة، إدارة الحالة، التفاعل مع المستخدم

### TypeScript
- **الإصدار:** 5.6.3
- **الوصف:** لغة برمجة تضيف أنواع البيانات الثابتة إلى JavaScript
- **الاستخدام:** كتابة الكود بشكل آمن مع التحقق من الأنواع

## قاعدة البيانات

### Prisma
- **الإصدار:** 5.0.0
- **الوصف:** ORM (Object-Relational Mapping) لإدارة قاعدة البيانات
- **الاستخدام:** تعريف النماذج، إدارة الهجرات، الاستعلامات

### PostgreSQL
- **الوصف:** نظام إدارة قواعد البيانات العلائقية
- **الاستخدام:** تخزين البيانات الأساسية للتطبيق

### Supabase
- **الإصدار:** @supabase/ssr 0.5.2, @supabase/supabase-js 2.39.0
- **الوصف:** منصة Backend-as-a-Service توفر قاعدة بيانات PostgreSQL وخدمات مصاحبة
- **الاستخدام:** قاعدة البيانات، المصادقة، التخزين

## التصميم والواجهة

### Tailwind CSS
- **الإصدار:** 3.4.13
- **الوصف:** إطار عمل CSS utility-first لتصميم الواجهات
- **الاستخدام:** تصميم جميع مكونات الواجهة، دعم RTL للغة العربية

### PostCSS
- **الإصدار:** 8.4.47
- **الوصف:** أداة لتحويل CSS باستخدام JavaScript plugins
- **الاستخدام:** معالجة ملفات Tailwind CSS

### Autoprefixer
- **الإصدار:** 10.4.20
- **الوصف:** أداة لإضافة بادئات المتصفحات تلقائياً
- **الاستخدام:** ضمان التوافق مع جميع المتصفحات

### tsx

## هيكل المشروع

```
nafes-training/
├── src/
│   ├── app/              # صفحات Next.js (App Router)
│   ├── components/       # مكونات React قابلة لإعادة الاستخدام
│   ├── lib/              # مكتبات ووظائف مساعدة
│   ├── types/            # تعريفات TypeScript
│   └── data/              # ملفات البيانات JSON
├── prisma/               # مخططات قاعدة البيانات
├── public/               # الملفات الثابتة
├── scripts/              # سكريبتات مساعدة
└── supabase/             # إعدادات Supabase
```

## أوامر التشغيل

```bash
# التطوير
npm run dev

# البناء للإنتاج
npm run build

# التشغيل بعد البناء
npm run start

# فحص الكود
npm run lint

# فحص قاعدة البيانات
npm run check-db
```

## ملاحظات مهمة

1. **TypeScript:** التطبيق يستخدم TypeScript بشكل كامل لضمان جودة الكود
2. **RTL Support:** التطبيق يدعم اللغة العربية بالكامل مع اتجاه RTL
3. **Server Components:** استخدام Next.js Server Components للأداء الأفضل
4. **API Routes:** استخدام Next.js API Routes لبناء واجهات برمجية
5. **Prisma Migrations:** استخدام Prisma لإدارة هجرات قاعدة البيانات
