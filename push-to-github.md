# رفع التغييرات إلى GitHub

افتحي **Terminal** أو **PowerShell** أو **Git Bash** من مجلد المشروع، ثم نفّذي الأوامر بالترتيب:

## 1. عرض الملفات المُعدّلة
```bash
git status
```

## 2. إضافة كل التغييرات
```bash
git add -A
```

## 3. إنشاء commit مع رسالة
```bash
git commit -m "إزالة القيم الافتراضية من صفحة الطالبة وتحديث الاسم من API"
```
أو لضم كل التغييرات السابقة في رسالة واحدة:
```bash
git commit -m "تحسينات: إرسال الاختبارات للطالبات، إزالة القيم الافتراضية، تشخيص عدم الظهور"
```

## 4. الرفع إلى GitHub
```bash
git push origin main
```

إذا كان الفرع الافتراضي عندك `master` بدل `main`:
```bash
git push origin master
```

---

### ملخص التغييرات المرفوعة
- **صفحة الطالبة:** إزالة القيم الافتراضية ("طالبة"، "—"، "لم تُسجّل بعد") وعرض الاسم الحقيقي أو "مرحبا، أهلاً بكِ"، وتحديث الاسم من API المهارات إن لم يكن في التخزين
- إصلاح إرسال الاختبار (shareToAll) في صفحتي النماذج التشخيصية ومشاركة الاختبارات
- منع الكاش لظهور الاختبارات المعينة عند الطالبات
- دعم البحث بالـ id أو رقم الطالبة في API الاختبارات المعينة
- زر تحديث وزر "لماذا لا أرى اختبارات؟" مع وضع تشخيص في API
- حماية صفحة اختيار الاختبار وتأكد صحة بيانات الطالبة (id)
- تحويل النماذج المخصصة إلى النماذج الجاهزة عند المشاركة
- تحديث سكربت جدول test_shares (TEST_SHARES_MIGRATION.sql)
